# Heart Disease Prediction ❤️  This project predicts whether a person is likely to have heart disease using Machine Learning techniques.  It analyzes various features such as age, sex, cholesterol levels, blood pressure, and other medical indicators to make accurate predictions.  The model is trained on a heart disease dataset and uses classification algorithms to predict risk.  ## ⚙️ Technologies Used - Python - Pandas - NumPy - Scikit-learn - Matplotlib / Seaborn - Google Colab  ## 📊 Model - Logistic Regression / Random Forest (choose whichever you used)  ## 📁 Dataset - Heart disease dataset (CSV format)  ## ▶️ How to Run 1. Open the notebook in Google Colab 2. Install required libraries (if needed) 3. Run all cells step by step
